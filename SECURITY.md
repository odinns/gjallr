# Security

Gjallr handles untrusted WordPress dumps, old site trees, and downloaded archive material. Treat all of it as hostile until proven boring.

## Supported Versions

Security fixes target the latest tagged release.

## Reporting

Report security issues privately through GitHub Security Advisories:

```text
https://github.com/odinns/gjallr/security/advisories/new
```

Include:

- affected version or commit
- reproduction steps
- impact
- any relevant sample input with secrets removed

Do not publish exploit details before there is a fix.
